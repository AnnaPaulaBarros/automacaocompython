Descrição do Projeto

O script utiliza Python e a biblioteca PyAutoGUI para interagir com o navegador e preencher formulários automaticamente. Ele lê os dados dos alunos a partir de um arquivo CSV e cadastra cada um no site definido.

Funcionalidades

Abrir o navegador automaticamente.

Navegar até o site de cadastro de alunos.

Realizar login no sistema.

Ler os dados dos alunos de um arquivo CSV (Nome, Email, Endereço, Telefone).

Inserir automaticamente os dados no formulário e submeter.

Repetir o processo para todos os alunos listados.

Tecnologias Utilizadas

Python 3

Bibliotecas: pyautogui, pandas, time, keyboard
